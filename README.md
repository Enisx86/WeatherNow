WeatherNow is a weather forecast application that displays the most important metorological informantion in real time. WeatherNow enables users to gain quick and detailed insight into current weather conditions, short-term and long-term forecasts, as well as additional meteorological information relevant to everyday activities.

The application is intended for a wide range of users for whom reliable weather information is essential for planning travel, outdoor work, sports activities, or daily obligations.

The main purpose of the application is to combine accurate meteorological data with a pleasant and intuditive user interface. WeatherNow provides information on temperature, fells-like temperature, air humidity, wind speed and direction, probability of precipitation, UV index, as well as hourly and daily forecasts for the selected location.

The application uses reliable external meteorological services (APIs) to ensure data accuracy, with a special focus on ease of use, fast response time, and visual clarity. The goal of the project is to develop a functional, scalable and easily extensible mobile application that can serve as a foundation for additional advanced features in the future.
